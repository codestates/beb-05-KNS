## Team

## Description
- NFT와 코인 정보를 공유하고 추천을 받으면 토큰을 보상으로 주는 토큰 인센티브 기반 커뮤니티 사이트입니다.
- 게시글을 올리거나 추천 10개를 받으면 토큰 보상을 받고 교환도 가능하며, 해당 토큰으로 발행된 NFT를 구매할 수 있습니다.
